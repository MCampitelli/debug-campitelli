1. Escreva como termina as palavras que deseja utilizar e utilize os checkbox para controlar a ativação dessas palavras;

2. Agora insira o texto para o qual deseja criar o debug, exemplo o select ou bloco da rotina;

3. Agora selecione qual é a saída desejada e aperte em Formatar.

4. Por fim, podes copiar o resultado obtido ou realizar o processo novamente
